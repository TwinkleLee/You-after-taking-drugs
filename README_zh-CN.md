# You after Taking Drugs

[English](README.md) | 简体中文

## Get Started

1. 下载`dlib`模型并放置到`model`文件夹下. [link](http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2)

2. `python face_distort.py` 运行demo并查看输出
